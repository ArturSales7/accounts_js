# 🏦 Accounts

Projeto para rodar um minibanco pelo terminal, salvando dados com JSON.

Você consegue criar uma conta, fazer login, depositar, sacar e consultar saldo, tudo persistido em um arquivo JSON usando o módulo `fs` nativo do Node.js.

---

## 🚀 Como rodar

```bash
# Clone o repositório
git clone https://github.com/ArturSales7/accounts_js

# Instale as dependências
npm install

# Rode o projeto
npm start
```

---

## ✨ Funcionalidades

- Criar conta com nome e senha
- Fazer login
- Depositar
- Sacar
- Consultar saldo

---

## 🛠️ Stack

| Tecnologia | Uso |
|---|---|
| Node.js | Ambiente de execução |
| fs (nativo) | Leitura e escrita de dados em JSON |
| Inquirer.js | Menus interativos no terminal |
| Chalk | Estilização e cores no terminal |
